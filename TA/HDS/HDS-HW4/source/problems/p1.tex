\section{مقدماتی}




برای مساله‌ی بهینه‌سازی زیر حکم‌های زیر را ثابت کنید:
\[
z_{opt}=\arg\min_{z_{n\times1}}\|y_{m\times1}-A_{m\times n}z_{n\times1}\|_{2}^{2}+\lambda\|z_{n\times1}\|_{1}.
\]
\begin{itemize}
\item 
نشان دهید کوچک ترین $\lambda$ که درایه‌های $z_{opt}$‌ همگی صفر شوند، برابر است با
\[
\lambda_{max}=\max_{j}\langle z_{j},y\rangle.
\]

\item
 فرض کنید برای مقدار مشخصی از 
$\lambda\ge0$
مساله بهینه سازی بالا منجر به دو جواب 
$\hat{z}_{1},\hat{z}_{2}$
شود. نشان دهید باید 
$A\hat{z}_{1}=A\hat{z}_{2}$

\item
با توجه به مفروضات قسمت ب، اگر 
$\lambda>0$
نشان دهید 
$\|\hat{z}_{1}\|_{1}=\|\hat{z}_{2}\|_{1}$.
\end{itemize}
